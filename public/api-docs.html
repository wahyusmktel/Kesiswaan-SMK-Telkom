Aplikasi Izin REST API

Dokumentasi API untuk pengembangan aplikasi mobile (Flutter)

Otentikasi

API menggunakan Laravel Sanctum untuk otentikasi. Gunakan token Bearer di header untuk setiap permintaan yang membutuhkan otentikasi.

Header: Authorization: Bearer {token}
GET /api/general/master-data

Mendapatkan data master (Kelas, Rombel, Tahun Pelajaran) untuk dropdown/filter.

Profil & Pengaturan

GET /api/profile

Mendapatkan detail profil lengkap pengguna (termasuk data Siswa/Guru).

POST /api/profile/update

Memperbarui informasi dasar profil (Nama, Email).

Parameter Body
Parameter Tipe Keterangan
name * string Nama lengkap.
email * string Email (harus unik).
POST /api/profile/update-password

Memperbarui kata sandi pengguna.

Parameter Body
Parameter Tipe Keterangan
current_password * string Kata sandi saat ini.
password * string Kata sandi baru.
password_confirmation * string Konfirmasi kata sandi baru.
POST /api/login

Gunakan endpoint ini untuk masuk dan mendapatkan token akses.

Parameter Body
Parameter Tipe Keterangan
email * string Email pengguna yang terdaftar.
password * string Kata sandi pengguna.
device_name string Nama perangkat (opsional).
Contoh Respon
{
  "success": true,
  "message": "Login berhasil",
  "data": {
    "token": "1|ABC...",
    "user": {
      "id": 1,
      "name": "Jane Doe",
      "email": "jane@example.com",
      "role": "siswa"
    }
  }
}

Siswa

GET /api/siswa/stats

Mendapatkan statistik poin dan perizinan siswa.

Contoh Respon
{
  "success": true,
  "data": {
    "points": 10,
    "status": {
      "label": "Aman",
      "color": "green"
    },
    "total_violations": 2,
    "total_achievements": 0,
    "total_permissions": 5
  }
}
GET /api/siswa/riwayat

Mendapatkan riwayat pelanggaran dan perizinan siswa.

POST /api/siswa/izin

Mengajukan permohonan izin baru.

Parameter Body (Multipart/form-data)
Parameter Tipe Keterangan
jenis_izin * string Sakit, Izin, atau Dispensasi.
keterangan * text Alasan mendetail.
tanggal_izin * date Format YYYY-MM-DD.
dokumen file Image atau PDF (max 2MB).
GET /api/siswa/exit-permits

Mendapatkan riwayat Izin Meninggalkan Kelas siswa.

POST /api/siswa/exit-permits

Mengajukan Izin Meninggalkan Kelas baru.

GET /api/siswa/bk/history

Mendapatkan riwayat pembinaan BK siswa.

Guru (Wali Kelas / Piket)

GET /api/guru/pending-approvals

Mendapatkan daftar permohonan izin yang menunggu persetujuan.

POST /api/guru/approve/{id}

Menyetujui atau menolak permohonan izin.

Parameter Body
Parameter Tipe Keterangan
status * string disetujui atau ditolak.
alasan_penolakan string Wajib jika status ditolak.
GET /api/guru/exit-permits/pending

Mendapatkan permohonan Izin Meninggalkan Kelas yang menunggu persetujuan guru.

POST /api/guru/exit-permits/approve/{id}

Menyetujui atau menolak Izin Meninggalkan Kelas.

BK Chat (Universal)

GET /api/chat/rooms

Mendapatkan daftar ruang obrolan BK.

GET /api/chat/messages/{roomId}

Mendapatkan pesan dalam ruang obrolan.

POST /api/chat/messages/{roomId}

Mengirim pesan baru.
